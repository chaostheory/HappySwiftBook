# Chapter 4
## Tuples

Tuples are

```javascript
var swiftStomach = ("lady bug", "moth", 5334)
```

How do you access parts of the tuple?

```javascript
var swiftStomach = ("lady bug", "moth", 5334)
print(swiftStomach.0)
print(swiftStomach.1)
```

![dissection](images/worm_stomach.jpg)

The result will be:

```javascript
"lady bug"
"moth"
```
